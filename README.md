# cpp_template
Create cpp template
<hr>
Quickly create a file with standard markup


```cpp
#include <iostream>

using namespace std; 
 
int main(int argc, char *argv[])
{

	return 0; 

}
```
**installation:**
>$ cd Downloads/

>$ git clone https://github.com/pekaby/cpp_template

>$ cd cpp_template

>$ sudo cp cpp /bin/

**usage:**
>$ cpp file_name.cpp

**Example:**

$ cpp main.cpp

or

$ cpp /mycppproject/mainfile/main.cpp
